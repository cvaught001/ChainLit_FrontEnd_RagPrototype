# Northstar Assistant

A Chainlit UI that connects to the Northstar RAG API. It sends user prompts to `/ai/query` and renders the model output in a clean, branded chat experience.

- Provides a lightweight chat UI for the Northstar LLM endpoint
- Reads API settings from environment variables
- Uses a custom avatar and light theme for branding
